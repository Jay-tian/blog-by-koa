# koa-symphony

## start
基于 koa2的框架。
采用三层架构，mvc思想，分离视图层和业务处。
集成了twig.js，webpack4，sequelize等组件。
方便了数据库查询，以及模版渲染

请使用 https://www.npmjs.com/package/cli-koa 生成项目
```
yarn global add cli-koa
cli-koa init <projectName>
```


## 目录及参数说明
```
config: 配置文件目录
config/db.json: 数据库配置文件
config/parameters.js：项目配置文件
```
